Портфолио

Бургерная
Одностраничный лендинг-пейдж с девятью секциями. Представляет собой 9 страниц с кнопкой перехода между окнами, красивым графическим слайдером и картой расположения офиса клиента. На сайте присутствует анимация. Применены технологии html, css, qjueru и native JS. Разработка проходила с помощью технологий SCSS и Gulp, редактор VS Code
